# iCombos
[cocos2d-x](https://github.com/cocos2d/cocos2d-x)

Download the code from [cocos2d download site](http://www.cocos2d-x.org/download/version#Cocos2d-x)
	
	cd cocos2d-x
	./setup.py
	source ~/.bash_profile
	cocos new iCombos -p com.suchachinoiserie.iCombos -l cpp -d ./
